HTML e CSS

Introdução a HTML e CSS

HTML (HyperText Markup Language) e CSS (Cascading Style Sheets) são duas das principais tecnologias usadas para construir a web. HTML fornece a estrutura da página, enquanto CSS é usado para estilizar a aparência da página.

HTML é uma linguagem de marcação que usa 'tags' para estruturar conteúdo em uma página web. Cada elemento de uma página web é encapsulado em uma tag HTML, que informa ao navegador como renderizar o conteúdo.

CSS é uma linguagem de estilo usada para descrever a aparência de elementos escritos em HTML. Com o CSS, você pode controlar coisas como cores, fontes, posicionamento de elementos, tamanhos, espaçamentos e muito mais.

Ao aprender HTML e CSS, você adquire as habilidades necessárias para criar e estilizar páginas web, permitindo que você construa sites do zero ou personalize sites existentes com seu próprio estilo.

Tags HTML

<h1> a <h6>

As tags de cabeçalho <h1> a <h6> são usadas para definir títulos e subtítulos. <h1> é o cabeçalho de maior nível e <h6> é o de menor nível.

<p>

A tag <p> é usada para definir um parágrafo.

<a>

A tag <a> é usada para criar um link. O atributo 'href' é usado para especificar o URL do link.

<img>

A tag <img> é usada para inserir uma imagem. Os atributos 'src' e 'alt' são usados para especificar o URL da imagem e o texto alternativo, respectivamente.

<div>

A tag <div> é usada como um contêiner para outros elementos HTML. É útil para agrupar elementos para aplicar estilos CSS ou realizar manipulações DOM com JavaScript.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a aparência e formatação de um documento escrito em HTML. CSS é usada para controlar o layout de várias páginas web de uma só vez.

Seletores

Os seletores são usados para selecionar o(s) elemento(s) HTML que você deseja estilizar.

Propriedades

As propriedades são usadas para declarar os estilos que você deseja aplicar ao elemento selecionado. Cada propriedade tem um valor.

Regras CSS

Uma regra CSS consiste em um seletor e uma declaração de bloco. O bloco de declaração contém uma ou mais declarações separadas por ponto e vírgula.

Incorporando CSS

CSS pode ser incorporado em HTML de três maneiras: Inline, Interno e Externo.

  • Inline: Os estilos são aplicados diretamente aos elementos HTML usando o atributo 'style'.
  • Interno: Os estilos são definidos no cabeçalho da página HTML dentro da tag 'style'.
  • Externo: Os estilos são definidos em um arquivo CSS externo que é vinculado à página HTML usando a tag 'link'.
